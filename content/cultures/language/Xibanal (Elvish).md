# High Elven Naming Canon

A compact naming framework for elven place names using:

- Hebrew triconsonantal roots as semantic cores
- Nahuatl-style suffixes for place and people forms
- Greek/Latin-style prestige affixes for imperial, civic, and ceremonial names

## Usage Pattern

Base pattern:

`[optional prestige prefix] + [surface form of Hebrew root] + [suffix]`

Examples:

- `Xal` + `hoph` + `tlan` -> `Xalhophtlan`
- `Tol` + `nahar` + `co` -> `Tolnaharco`
- `Aure` + `malak` + `ion` -> `Auremalakion`

## Table 2. Nahuatl-Style Suffixes

| Suffix | Meaning / Function | Usage |
|---|---|---|
| -tlan | place of, land of | Regions, realms, homelands |
| -co | at, in, city of | Cities, ports, capitals |
| -pan | upon, above, on the high place | Highlands, terraces, elevated regions |
| -teca | people of | Ethnic group, cultural people |
| -tecatl | person of | Singular inhabitant or noble demonym |
| -yan | place where | Functional locations, ritual or trade sites |
| -can | place, locality | Minor place names |
| -nal | associated with, characterized by | Descriptive adjectival or regional form |

## Notes

- Surface forms should be smoothed for pronunciation before canonization.
- Do not feel obligated to use every root literally; this is a semantic seedbank.
- Keep names between 2 and 5 syllables where possible.
- Prefer repeated family patterns within a region so names feel historically related.
